# XOR File Processor

Десктопное приложение на Qt/C++ для обработки файлов операцией XOR.

## Описание

Программа мониторит указанную папку, применяет побитовую операцию XOR к содержимому найденных файлов и сохраняет результат в папку назначения. Поддерживает как разовый запуск, так и периодический опрос по таймеру.

## Функциональность

- Фильтрация входных файлов по маске
- XOR-обработка файлов с 8 байтным ключом
- Два режима запуска: разовый и по таймеру с настраиваемым периодом
- Настраиваемое поведение при совпадении имён выходных файлов: перезапись или создание копии с счётчиком
- Опциональное удаление входных файлов после обработки
- Обработка файлов в отдельном потоке GUI не блокируется
- Прогресс-бар и лог с временными метками

## Использование

1. Указать маску входных файлов (например `*.bin`)
2. Выбрать папку входных файлов кнопкой **Обзор...**
3. Выбрать папку для сохранения результатов
4. Ввести XOR ключ в HEX формате, ровно 16 символов
5. Выбрать режим запуска: **Разовый** или **Таймер**
6. При режиме таймера указать период опроса в секундах
7. Нажать **Старт**
